class Factory¶
#include <factory.h>

A class to create polymorphic objects by means of keys.

Factories serve to instantiate a derived class using a key that is known only at run-time. It is a singleton class. One can register a key with a derived type in three different forms:

  1. in the factory’s constructor;
  2. by accessing the singleton instance directly; or
  3. using a helper register class

Template Parameters
  • The: Factory type (CRTP)
  • Base: The base class of the polymorphic type
  • Key: The type of the keys
  • Comp: The type of the comparator used in the ordering of the keys
  • Args: The types of the arguments passed as parameters to the object constructor
