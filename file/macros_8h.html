CPPORM_REGISTER(factory, derived, varname, keys...)¶

Register type with a key in factory.

Usage example:

CPPORM_REGISTER(SerializerFactory, cpporm::Serializer, cpporm, "cpporm");

CPPORM_MAP_ATTRIBUTE(ent, att)¶

Map an attribute name to an attribute object.

Usage example:

static const cpporm::AttributeMap cMap = {CPPORM_MAP_ATTRIBUTE(MyEntity, attr)};

CPPORM_MAP_INDEX(ent, ind)¶

Map an index name to an index object.

Usage example:

static IndexMap map = {CPPORM_MAP_INDEX(MyEntity, primary_key)};

CPPORM_MAP_RELATIONSHIP(ent, rel)¶
