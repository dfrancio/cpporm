Class cpporm::__cxx11::Relationship¶

class cpporm::__cxx11::Relationship¶

Abstract class to represent database relationships.

A relationship has a name, a set of properties, a list of names of foreign keys and a list of names of referenced columns (one for each foreign key). Relationships should derive from ToOneRelationship or ToManyRelationship, not from this class.
